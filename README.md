# Python医疗数据分析可视化实时监控系统

## 核心功能
- 前后端分离设计
- Python语言和Flask框架后端开发
- Vue框架前端用户交互和数据可视化
- MySQL数据库存储和管理医辽文本数据
- 数据预处理（文本清洗、分词、特征提取）
- 随机森林分类算法模型训练和预测
- 可视化展示病预测结果和数据分布

## 适用场景
- 医疗数据分析
- 病预测
- 医学文献和病历记录分析
- 医疗决策支持

## 技术特点
- 前后端分离架构，提高开发效率和系统可维护性
- Python和Flask框架提供稳定的数据处理和算法支持
- Vue框架实现用户友好的交互界面和可视化展示
- 随机森林算法模型提供准确的病预测能力
